About EVE Frontier Modding

Under the theme “A Toolkit for Civilization,” this Hackathon invites builders to create the infrastructure, systems, and tools that a recovering civilization might depend on, and to deploy them directly into a live universe.

EVE Frontier is designed around extensibility. At the core of this design are Smart Assemblies: programmable, player-built structures such as Storage Units, Gates, and Turrets that exist physically within the game world. These structures are not static assets. They are capable of hosting custom logic, defining access rules, enforcing behaviors, and interacting with other systems across the galaxy.

Builders participating in the Hackathon can create mods that run within these Smart Assemblies, introducing new mechanics and persistent functionality into the shared environment. Because these assemblies exist in space, they can be interacted with, defended, contested, or destroyed, ensuring that player-built systems remain embedded within the fabric of the world itself.

In addition to in-world extensions, participants may also build external applications that connect to EVE Frontier through the World API. These tools can include maps, dashboards, coordination platforms, analytics services, or entirely new interfaces for interacting with the universe. While these applications do not exist as objects in space, they operate against live data, responding to and reflecting the evolving state of the Frontier in real time.
